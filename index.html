Face Detection Application

Face Detection is an application that I've made watching "The Complete Web Developer: Zero to Mastery" course by Andrei Neagoie.

It is a fully responsive website created using React.js library.

Pokemen Battle Application

Lights Out

Lights Out Game

Lights Out

Lights Out - Logic Game

Lights Out Game is an application that I've made watching "The Modern React Bootcamp" course by Colt Steele. It is a fully responsive website created using React.js library. This project is based on an old handheld game put out in the 80s - early 90s.

Lights Out is a logic (puzzle) game, played on a grid of individual lights, which can either be lit or unlit. The puzzle is won when all of the lights are turned off. A user can click on a cell to toggle that light - but it also toggles the light above it, to the left of it, to the right of it, and below it. Cells on an edge or in the corner won't flip as many lights, since they are missing some neighbors.

The goal is to turn all of the lights out. Therefore, if a user will turn them all off, they get a message "You Won!" on the main page instead of the game board.

  • Technologies Used
  • JS, React, JSX, HTML, CSS
  • https://juliagood.github.io/lights-out-game/
